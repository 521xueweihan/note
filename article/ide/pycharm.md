## pycharm快捷键
**注：如mac下无效，ctrl替换成commond**  
- cmd + Numpad+/- 展开/折叠代码块（当前位置的：函数，注释等）--加shift是全部
- 分屏 你对tab点右键看咯
- 跳转／返回 类和方法的应用部分 cmd + '['或']'
- 调试(Debugging)
	- F8   跳过
	- F7   进入
	- Shift + F8   退出
	- Alt + F9    运行游标
	- Alt + F8    验证表达式
	- Ctrl + Alt + F8   快速验证表达式
	- F9    恢复程序
	- Ctrl + F8   断点开关
	- Ctrl + Shift + F8   查看断点
